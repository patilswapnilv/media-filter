Filter media files by author.

== Description ==

Media Filter adds image width and height, clickable author link, 'mine' link, PDF and ZIP links in Media Library (upload.php). 
It makes media files easier to find in multi-author websites.


== Installation ==

1. Upload `media-filter` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why was this plugin created? =

This is a great help in multi-author websites where you have lots of media files.

= Why there is no 'Mine' link in my Media Library? =

Mine link is only shown when there are more than one user and user have attachments.
